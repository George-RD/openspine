# Vendored skills: finding-unknowns pack

The eight skill directories here are vendored from
[Neeeophytee/finding-unknowns-skills](https://github.com/Neeeophytee/finding-unknowns-skills)
(MIT-licensed skill text), a community distillation of Thariq Shihipar's essay
*A Field Guide to Fable: Finding Your Unknowns*
(<https://thariqs.github.io/html-effectiveness/unknowns/>). The underlying techniques
belong to their author; this README carries the attribution.

Repo-scoped install: OMP discovers these as `<repo>/.omp/skills/<name>/SKILL.md`, so they
load only for this repository. They assist the spec rounds — blindspot passes over the
design log, interview-style decision closing, implementation plans/notes, and pre-merge
change quizzes. First applied in the 2026-07-07 blindspot round (see
`.raw/openspine-agentos-design-log.md`, AD-137 / OQ-14..18 provenance).

Update: re-vendor from upstream; do not edit skill text in place.
